# Formulario de registro para nuevos usuarios
```
Algoritmo Formulario_de_registro
	
	Imprimir "---FORMULARIO DE USUARIO---"
	Escribir "Nombre"
	Leer Nombre
	Escribir "Apellido"
	Leer Apellido
	Escribir "Edad"
	Leer Edad
	Escribir "Correo electronico"
	Leer Correoelectronico
	Escribir "Dirección"
	Leer Dirección
	
	Imprimir "---DATOS DEL USUARIO---"
	Imprimir Nombre
	Imprimir Apellido
	Imprimir Edad
	Imprimir Correoelectronico
	Imprimir Dirección
	
FinAlgoritmo
```
